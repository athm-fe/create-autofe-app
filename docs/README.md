## Getting Started

快速创建项目 :tada:

```shell
npx create-autofe-app my-app

cd my-app
npm start
```

或者先安装 `create-autofe-app`

```shell
npm install -g create-autofe-app
# OR
yarn global add create-autofe-app
```

然后创建项目

```shell
create-autofe-app my-app

cd my-app
npm start
```
